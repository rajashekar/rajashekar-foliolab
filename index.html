WakeWordDetector

WakeWordDetector is an open-source project designed to create a custom wake word detector. It aims to detect specific audio sequences (wake words) like "Ok, Google" to trigger actions in applications. The project includes detailed implementation steps from dataset preparation, feature extraction, model training, and testing, to inference using various methods like PyAudio, web sockets, ONNX, TensorFlow.js, and TFLite. The project references existing wake word detection systems like Firefox Voice and draws upon concepts of data pre-processing, audio feature extractions and model architectures for effective wake word detection.

ethnicolr

ethnicolr is a Python package that predicts an individual's race and ethnicity based on their name (first and last, or just last). It leverages US census data, Florida voting registration data, and Wikipedia data. The predictions offer varying levels of granularity depending on the utilized dataset, ranging from broad categories (e.g., Non-Hispanic White) to more specific ethnic groups. The package aims to provide insights into ethnicity and race using the model.
