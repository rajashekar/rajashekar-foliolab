layoutlens

LayoutLens is an AI-powered tool designed to assist in UI testing. It leverages artificial intelligence to automatically detect and analyze layout issues, ensuring a consistent and visually appealing user experience across different devices and screen sizes. The tool aims to streamline the testing process, reducing the manual effort required to identify and address layout problems, ultimately helping improve the quality of web and mobile applications.

reposum

This project, reposum, automatically generates a list of repositories within an organization, summarizes each repository using OpenAI, and then publishes the summaries online. It streamlines the process of understanding a large number of repositories by providing concise overviews, making it easier to keep track of projects and their purpose. This allows for quick understanding of the project and its functions without the need to manually visit each project.

diverse_stories

This project utilizes GenAI to adapt classic stories and novels, incorporating diverse characters and cultural contexts. The goal is to reimagine existing narratives, broadening representation and making them more inclusive for modern audiences. The project focuses on working with stories in the public domain to avoid copyright issues and leveraging the power of AI to rewrite and enhance the original content to reflect a wider range of experiences.
